# BattleOfOlympo
Battle of Olympo é um jogo de duelo tático em turnos para terminal, desenvolvido em Python. O projeto é uma releitura temática do jogo de tabuleiro e dados Super Smash Dice, adaptando suas mecânicas para a rica mitologia grega com um sistema de combate focado em peso e estabilidade.
Você pode jogar o Battle of Olympo diretamente no seu navegador através do OnlineGDB:
https://onlinegdb.com/hIjJMH4D1
